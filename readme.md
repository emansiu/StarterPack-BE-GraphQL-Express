# Starter Pack
## Back end server with GraphQL and Express

### included dependancies
* express
* graphql
* express-graphql

### dev dependancy
* nodemon

### scripts added to package.json
```json
"scripts":{
    "devStart":"nodemon server.js"
}
```

## Get Started
Just run:
npm run devStart

This is will get your server up and running on localhost:5000
